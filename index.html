Project Overview

The Online Library Management System is a desktop-based application that allows users to browse, search, borrow, and return media items such as books and films. The system tracks availability, borrower information, and due dates while providing clear visual feedback.

Technologies Used

  • Python – Core programming language
  • Flask – RESTful backend API
  • PyQt5 – Desktop graphical user interface
  • JSON – Persistent data storage
  • Git & GitHub – Version control
  • GitHub Pages – Project documentation website

Key Features

  • Browse media by category
  • Search media using partial name matching
  • Borrow and return functionality
  • Borrower name and due date tracking
  • Automatic overdue detection
  • Visual indicators for borrowed items
  • Media cover image support
  • Dark mode in the desktop application

System Architecture

The system follows a three-layer architecture to ensure maintainability and separation of concerns:

  • Frontend (PyQt5): Handles user interaction and presentation logic.
  • Backend (Flask): Provides a RESTful API responsible for business logic, validation, and data processing.
  • Data Layer (JSON): Stores media information persistently in a lightweight, human-readable format.
