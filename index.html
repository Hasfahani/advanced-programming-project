Project Overview

The Online Library Management System is a desktop-based application that allows users to browse, search, borrow, and return media items such as books and films. The system tracks availability, borrower information, and due dates, providing clear visual feedback to the user.

Technologies Used

  • Python – Core programming language
  • Flask – RESTful backend API
  • PyQt5 – Desktop graphical user interface
  • JSON – Lightweight persistent data storage
  • Git & GitHub – Version control
  • GitHub Pages – Project documentation website

Key Features

  • Browse media by category
  • Search using partial name matching
  • Borrow and return media items
  • Borrower name and due date tracking
  • Automatic overdue detection
  • Visual indicators for borrowed items
  • Media cover image support
  • Dark mode in the desktop application

System Architecture

The system follows a clear three-layer architecture to ensure maintainability and separation of concerns:

  • Frontend (PyQt5): Handles user interaction and presentation logic.
  • Backend (Flask): Provides a REST API responsible for business logic and validation.
  • Data Layer (JSON): Stores media records persistently in a human-readable format.
